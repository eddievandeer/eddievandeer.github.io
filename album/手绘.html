个人的一些手绘作品

2020-08-09
0  -  0m
画册
绘画
铅笔
ACG

# 《火影忍者》宇智波佐助

约摸2016年，记得瞎画了不少东西，但留下来的就这几张了

第一次画动漫人物，算是黑历史了

# 《火影忍者》旗木卡卡西

临摹的原图是水墨风格的，当时看觉得还行，现在再看。。。黑历史+1

（图丢了，有空回家再找找）

# 《火影忍者》宇智波鼬

这应该是我初学阶段画的最好的了，记得当时磨了好久才画完

这会儿应该是高三了，没什么时间画画

# 《埃罗芒阿老师》和泉纱雾

万恶之源，从这张开始画的基本都是妹子了

接下来好像是高三的暑假，当时一时兴起在网上买了一套马克笔，高三的暑假嘛，没什么事做，闲来无事就拿着笔瞎画

# 《Re：从零开始的异世界生活》雷姆-1

又一个黑历史。。。第一次尝试马克笔上色，直接用圆珠笔画线稿，上头发的时候还拿错笔了，各种崩

# 《约会大作战》时崎狂三

头发颜色稍微深了点，线条都看不到了；大面积上色时，颜色十分不均匀

# 《为美好的世界献上祝福》阿库娅

逐渐会用马克笔了，但看着还是像抹鼻涕；才发现没上阴影。。。

# 《Re：从零开始的异世界生活》雷姆-2

又回到了铅笔画，感觉画功有那么点进步了，这张也算是个人比较满意的一张

# 《干物妹！小埋》土间埋

用马克笔画的最好的一次，也是最后一次用马克笔。。。后面都把马克笔送给小孩子当玩具了

高中生涯正式结束，步入大学

# 《碧蓝之海》古手川千纱

上大学后画的第一张画，是在上英语课的时候拿作业本的背面画的，原画已经不知道放哪了，只剩下这一张照片（还好拍了照不然亏死）

# 礼物

大一那年的圣诞节，画来送妹子的，画了三四天；最底下的那朵玫瑰由于时间有限，没仔细画，最后草草的收尾了。。。

# 《擅长捉弄的高木同学》高木-1

这部狗粮番真的是太香了，大爱高木

大一结束，进入暑假

# 《擅长捉弄的高木同学》高木-2

买了本速写本，第一页画上了最喜欢的高木；高木-1是漫画里的，这是动画里的，画风稍许有些不同

# 《Re：从零开始的异世界生活》雷姆-3

又是雷姆，哈哈！不过这张是高中没画完的，上了大学才继续画完；由于只用了自动铅笔，衣服颜色不够深，更不用说衣服的阴影了

暑假结束，返校，进入大二

# 《怪物猎人》雷狼龙

第一次尝试非人的生物，主要是为了庆祝怪猎冰原出雷狼龙了，当初P3最喜欢的怪

# 《怪物猎人》轰龙

回归的老怪之一，龙车印象最为深刻；感觉用铅笔很难画，一个一个去画鳞片是不可能的了，只能用上色代替

大二上结束，疫情要来了

# 《Fate系列》阿比盖尔

疫情爆发前，在学校的自习室画了线稿，回家后才上色，只用了自动铅笔，感觉这已经是极限了。。。

大三返校后

回学校后的一节英语课，拿起3B、4B和6B的铅笔修了修，稍微增强了些对比度，看起来顺眼了不少

# 《Fate系列》伊莉雅

说实话这张不太满意，脸没画好；由于同样是只用自动铅笔，上色方面也不太理想

大三返校后

又是一节英语课，再次掏出3B、4B和6B的铅笔，感觉修不回来了。。。

大三上疫情结束，返校

# 《擅长捉弄的高木同学》高木-3

刚开学就发烧了，还被学校强制隔离，一人一间房，住得实在是太无聊了，画个高木来陪我视频排解寂寞~~

# 《Fate系列》伊莉雅

好久没有画这么大一副的画了，虽说画大了好处理细节，但大面积的排起线来真的要命

# 《怪物猎人》麒麟套（女套）

这幅直接走了另一个极端了。。。关键是我开始画的时候都没意识到自己画的多小，画完轮廓发现这脸还没有我指甲盖大，就这样我还是硬给它画完了，经典微操

# 《怪物猎人》4代看板娘索菲亚

怪物猎人历代看板娘里最喜欢的就是索菲亚了，只可惜她只喜欢布拉基迪欧斯（碎龙）

说起来最近比较喜欢尝试画这种全身的，但经常处理不好比例，这一副刚开始画线稿的时候腿短了很多，还是在画完了一段时间才反应过来，看来还是得多练，把人体结构比例啥的都练熟，不然都不敢说自己会画画
